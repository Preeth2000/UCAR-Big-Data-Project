*Ensure MATLAB has parallel processing package installed

To run
1) Open SimpleEnsemble.m MATLAB file
2) Press run
3) A prompt  will appear asking for a directory, Choose the "24 hour" directory folder
4) Another prompt will appear asking for another directory, Choose the "24 hour original" directory folder
5) Code will run on its own

Note: Creating a parallel pool may take some time depending on the machine
      Code can take up to a minute to process
      6 figures should appear after it is run, each figure is labelled showing what is being displayed
